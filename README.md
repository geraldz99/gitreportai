##  Getting Started

###  Prerequisites

Before getting started with gitreportai, ensure your runtime environment meets the following requirements:

- **Programming Language:** TypeScript
- **Package Manager:** Pnpm


###  Installation

Install gitreportai using one of the following methods:

**Build from source:**

1. Clone the gitreportai repository:
```sh
❯ git clone https://github.com/geraldz99/gitreportai
```

2. Navigate to the project directory:
```sh
❯ cd gitreportai
```

3. Install the project dependencies:


**Using `pnpm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/pnpm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://pnpm.io/installation)

```sh
❯ pnpm install
```

###  Usage
Run gitreportai using the following command:
**Using `pnpm`** &nbsp; [<img align="center" src="https://img.shields.io/badge/pnpm-CB3837.svg?style={badge_style}&logo=npm&logoColor=white" />](https://pnpm.io/installation)

```sh
❯ pnpm run dev
```
